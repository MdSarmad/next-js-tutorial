To Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.


Project Structure:

- `app/page.tsx`: The main page of the application.
- `app/layout.tsx`: The layout of the application.
- `public/`: The public directory.
- `next.config.js`: The Next.js configuration file.
- `package.json`: The package.json file.
- `tsconfig.json`: The TypeScript configuration file.
- `README.md`: The README file.
- `next-env.d.ts`: The Next.js environment file.
- `postcss.config.js`: The PostCSS configuration file.
- `eslint.config.js`: The ESLint configuration file.
- `.next/`: The Next.js cache directory. It's created when you run `npm run dev`.
